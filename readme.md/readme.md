Propósito del proyecto: Automatizar pruebas de UI y de API para el sitio SauceDemo, aplicando prácticas como Page Object Model, manejo de datos externos, generación de reportes HTML, logging y captura automática de pantalla.

Tecnologías utilizadas:

    Python 3.x

    Pytest

    Selenium WebDriver

    Logging

    Faker

    CSV / JSON

    Request

Reportes y Logs: El proyecto genera tres tipos principales de resultados: reporte HTML, capturas de pantalla, archivo de log.

Reporte HTML: Se genera un reporte detallado con el nombre de reporte.html en la carpeta raíz del proyecto.

Propósito: Automatizar pruebas de UI y API para SauceDemo, usando Page Object Model, datos externos, reportes HTML, logging y capturas de pantalla.

Tecnologías: Python 3.x, Pytest, Selenium WebDriver, Logging, Faker, CSV/JSON y Request.

Resultados de ejecución:

    Reporte HTML: Archivo reporte.html generado en la raíz.

    Logs de ejecución: Información detallada en logs/suite.log.

    Capturas de pantalla: Generadas automáticamente durante las pruebas.

Ejecución: Para iniciar las pruebas, se utiliza el comando:
python -m run_test.py
